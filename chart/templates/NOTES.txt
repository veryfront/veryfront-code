To check the status:
  kubectl get pods -l app.kubernetes.io/instance={{ .Release.Name }} -n {{ .Release.Namespace }}

To scale components independently:
  # Scale proxy
  kubectl scale deployment {{ include "veryfront-renderer.fullname" . }}-proxy --replicas=<count> -n {{ .Release.Namespace }}

  # Scale renderer
  kubectl scale deployment {{ include "veryfront-renderer.fullname" . }}-renderer --replicas=<count> -n {{ .Release.Namespace }}

Or via Helm values:
  helm upgrade {{ .Release.Name }} ./chart \
    --set proxy.replicaCount=<count> \
    --set renderer.replicaCount=<count>
