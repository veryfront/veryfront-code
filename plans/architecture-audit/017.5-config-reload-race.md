# 017.5: Config Reload Partial Read

## Severity: MEDIUM

## Location

**File:** `src/core/config/loader.ts`

```typescript
// VULNERABLE - No file locking or atomic read
async function reloadConfig(): Promise<Config> {
  const content = await Deno.readTextFile(CONFIG_PATH);
  return JSON.parse(content);
}
```

## The Problem

File reads during partial writes can return incomplete or malformed content. Hot reload triggers can catch config files mid-write.

### Race Window

```
Config file being written:
{"port": 3000, "debug": tru   ← Write in progress

Concurrent read sees:
{"port": 3000, "debug": tru   ← Truncated/invalid JSON

Result: JSON.parse throws, config reload fails
```

### HMR Trigger Scenario

```
t0: User saves config file
t1: Editor writes first half of file
t2: File watcher detects change → triggers reload
t3: Config loader reads incomplete file
t4: JSON.parse fails
t5: Editor writes second half
t6: Config stays at old value OR error state
```

## Proposed Fix

### Option A: Atomic File Write Pattern

Ensure writers use atomic replacement:

```typescript
// Writer side - atomic replacement
async function writeConfig(config: Config): Promise<void> {
  const tempPath = `${CONFIG_PATH}.tmp.${Date.now()}`;
  const content = JSON.stringify(config, null, 2);

  // Write to temp file
  await Deno.writeTextFile(tempPath, content);

  // Atomic rename
  await Deno.rename(tempPath, CONFIG_PATH);
}
```

### Option B: Read with Retry

```typescript
async function reloadConfig(maxRetries = 3): Promise<Config> {
  for (let attempt = 0; attempt < maxRetries; attempt++) {
    try {
      const content = await Deno.readTextFile(CONFIG_PATH);
      return JSON.parse(content);
    } catch (error) {
      if (attempt < maxRetries - 1) {
        // Wait for write to complete
        await new Promise(resolve => setTimeout(resolve, 50));
        continue;
      }
      throw error;
    }
  }
  throw new Error("Config reload failed after retries");
}
```

### Option C: File Lock (Platform-Specific)

```typescript
import { flock } from "npm:fs-ext";

async function reloadConfigWithLock(): Promise<Config> {
  const file = await Deno.open(CONFIG_PATH, { read: true });
  try {
    // Acquire shared lock (blocks if exclusive lock held)
    await flock(file.rid, "sh");

    const content = await readAll(file);
    return JSON.parse(new TextDecoder().decode(content));
  } finally {
    await flock(file.rid, "un");
    file.close();
  }
}
```

### Option D: Debounce Reload

```typescript
let reloadTimer: number | null = null;
let lastConfig: Config;

function scheduleReload(): void {
  if (reloadTimer) {
    clearTimeout(reloadTimer);
  }

  // Wait for file writes to settle
  reloadTimer = setTimeout(async () => {
    reloadTimer = null;
    try {
      lastConfig = await reloadConfig();
    } catch (error) {
      console.warn("Config reload failed, keeping previous config");
    }
  }, 100);
}
```
