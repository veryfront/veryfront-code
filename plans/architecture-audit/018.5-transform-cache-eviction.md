# 018.5: Transform Cache No Eviction

## Severity: MEDIUM

## Location

**File:** `src/build/transforms/`

```typescript
// LEAK - Transform results cached forever
const transformCache = new Map<string, TransformResult>();

async function transform(input: TransformInput): Promise<TransformResult> {
  const key = buildCacheKey(input);

  if (transformCache.has(key)) {
    return transformCache.get(key)!;
  }

  const result = await executeTransform(input);
  transformCache.set(key, result);  // Never evicted
  return result;
}
```

## The Problem

Transform cache stores compiled output for every unique input. With content-addressed keys, any code change creates a new entry. Old entries are never removed.

### Growth Model

```
Per transform result:
├── Source code: 5-50KB
├── Compiled output: 5-50KB
├── Source map: 10-100KB
└── Total: 20-200KB

Development session (100 saves):
→ 100 cache entries × 100KB = 10MB

Multi-tenant production:
├── 100 projects × 200 files × 5 versions
├── = 100,000 entries
└── = 2-20GB cached transforms
```

### Version Churn

```
File: components/Button.tsx
├── v1: transformed, cached (key: hash1)
├── v2: transformed, cached (key: hash2)
├── v3: transformed, cached (key: hash3)
└── v1, v2 never accessed again, never evicted
```

## Proposed Fix

### Option A: LRU with Size Limit (Recommended)

```typescript
import { LRUCache } from "lru-cache";

const transformCache = new LRUCache<string, TransformResult>({
  max: 10000,  // Max entries
  maxSize: 1024 * 1024 * 1024,  // 1GB max
  sizeCalculation: (result) => {
    return (result.code?.length ?? 0) +
           (result.map?.length ?? 0) +
           (result.source?.length ?? 0);
  },
  ttl: 1000 * 60 * 60 * 4,  // 4 hour TTL
  updateAgeOnGet: true,
});
```

### Option B: Two-Tier Cache

```typescript
// Hot cache (in-memory, small)
const hotCache = new LRUCache<string, TransformResult>({ max: 1000 });

// Cold cache (disk/redis, large)
const coldCache = new DiskCache<TransformResult>("./cache/transforms");

async function getTransform(key: string): Promise<TransformResult | null> {
  // Check hot cache
  const hot = hotCache.get(key);
  if (hot) return hot;

  // Check cold cache
  const cold = await coldCache.get(key);
  if (cold) {
    hotCache.set(key, cold);  // Promote to hot
    return cold;
  }

  return null;
}

async function setTransform(key: string, result: TransformResult): Promise<void> {
  hotCache.set(key, result);
  await coldCache.set(key, result);
}
```

### Option C: Project-Scoped with Cleanup

```typescript
interface ProjectTransformCache {
  cache: LRUCache<string, TransformResult>;
  lastAccess: number;
}

const projectCaches = new Map<string, ProjectTransformCache>();

// Cleanup idle project caches
setInterval(() => {
  const now = Date.now();
  const IDLE_THRESHOLD = 30 * 60 * 1000; // 30 minutes

  for (const [projectId, cache] of projectCaches) {
    if (now - cache.lastAccess > IDLE_THRESHOLD) {
      projectCaches.delete(projectId);
    }
  }
}, 5 * 60 * 1000);
```
