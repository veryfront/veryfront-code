# 017.1: Memoize Cache Stampede

## Severity: HIGH

## Location

**File:** `src/utils/memoize.ts`
**Lines:** 35-51

```typescript
export function memoize<T>(fn: () => Promise<T>): () => Promise<T> {
  let cached: T | undefined;

  return async () => {
    if (cached === undefined) {
      cached = await fn();  // RACE: Multiple calls hit this
    }
    return cached;
  };
}
```

## The Problem

No in-flight deduplication. When cache is empty, N concurrent calls all execute the expensive function instead of waiting for the first.

### Thundering Herd Scenario

```
t0: Request A calls memoize() → cache empty → starts fn()
t1: Request B calls memoize() → cache empty → starts fn()
t2: Request C calls memoize() → cache empty → starts fn()
...
t10: 100 requests all executing fn() simultaneously

If fn() is a database query:
→ 100 identical queries instead of 1
→ Database overload
→ Cascading timeouts
```

### Current vs Expected Behavior

```
CURRENT (broken):
├── Cache empty
├── 100 concurrent requests
├── 100 fn() executions
└── 100x resource usage

EXPECTED (with dedup):
├── Cache empty
├── 100 concurrent requests
├── 1 fn() execution
├── 99 requests wait for result
└── 1x resource usage
```

## Proposed Fix

### In-Flight Deduplication

```typescript
export function memoize<T>(fn: () => Promise<T>): () => Promise<T> {
  let cached: T | undefined;
  let inFlight: Promise<T> | undefined;

  return async () => {
    // Return cached value
    if (cached !== undefined) {
      return cached;
    }

    // Join existing in-flight request
    if (inFlight !== undefined) {
      return inFlight;
    }

    // Start new request
    inFlight = fn().then((result) => {
      cached = result;
      inFlight = undefined;
      return result;
    }).catch((error) => {
      inFlight = undefined;
      throw error;
    });

    return inFlight;
  };
}
```

### With Key-Based Memoization

```typescript
export function memoizeWithKey<K, V>(
  fn: (key: K) => Promise<V>
): (key: K) => Promise<V> {
  const cache = new Map<K, V>();
  const inFlight = new Map<K, Promise<V>>();

  return async (key: K) => {
    if (cache.has(key)) {
      return cache.get(key)!;
    }

    if (inFlight.has(key)) {
      return inFlight.get(key)!;
    }

    const promise = fn(key).then((result) => {
      cache.set(key, result);
      inFlight.delete(key);
      return result;
    }).catch((error) => {
      inFlight.delete(key);
      throw error;
    });

    inFlight.set(key, promise);
    return promise;
  };
}
```
