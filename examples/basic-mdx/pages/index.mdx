# Welcome to Veryfront

This is a basic example of a Veryfront site using MDX.

## Features Demonstrated

- MDX page rendering
- React component integration
- File-based routing

## Try It Out

1. Edit this file (`pages/index.mdx`) and see hot reload in action
2. Create new pages by adding `.mdx` files to the `pages/` directory
3. Navigate to the [About page](/about) to see layouts in action
