# Benchmark Apps

This directory tracks the framework-specific app lanes used for apples-to-apples
benchmarking.

Each framework gets its own subdirectory with:

- `manifest.json` — current implementation status and scenario mapping
- framework-specific setup notes
- eventually, executable local benchmark apps

Phase 1 status:

- `veryfront/` — implemented via the repo-root temp-fixture benchmark harness
- `nextjs/` — implemented as a local Pages Router benchmark app
