## [0.1.0] - 2025-11-18

### Added
- **AI Native Framework**: Complete agent runtime with auto-discovery, tools, and MCP integration.
- **Rendering Engines**: Support for SSR, SSG, ISR, and experimental RSC.
- **Multi-Runtime Support**: Adapters for Deno, Node.js, Bun, and Cloudflare Workers.
- **Routing**: File-based routing supporting both App Router and Pages Router patterns.
- **Data Fetching**: Integrated `getServerData` and `getStaticData` APIs.
- **Documentation**: Comprehensive guides for routing, rendering, and AI features.
