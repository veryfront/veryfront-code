# Component Guides

Comprehensive guides for Veryfront's built-in components. These guides provide detailed examples, use cases, and best practices for each component.

## Available Guides

### [Head Component](./head.md)
Manage document metadata, SEO tags, and head elements. Learn how to:
- Set page titles and meta descriptions
- Configure Open Graph and Twitter Card tags
- Add favicons and canonical URLs
- Handle dynamic metadata based on routes

### [Image Component](./image.md)
Optimize and serve images efficiently. Learn how to:
- Use responsive images with automatic srcset generation
- Lazy load images for better performance
- Handle different image formats and sizes
- Implement blur placeholders and loading states

### [Link Component](./link.md)
Create navigation links with client-side routing. Learn how to:
- Navigate between pages without full page reloads
- Prefetch linked pages for instant navigation
- Style active links based on current route
- Handle external vs. internal links

### [Script Component](./script.md)
Load and manage external scripts safely. Learn how to:
- Load third-party scripts with optimal timing
- Handle script loading strategies (defer, async)
- Manage script dependencies and execution order
- Integrate analytics and tracking scripts
