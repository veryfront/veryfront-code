## Getting started

We are using the Inter open source font by Rasmus Andersson at Veryfront.
Inter version 4.0 was just released after being in the works for more than 2 years.
The font can be found on the [GitHub Release Page](https://github.com/rsms/inter/releases/tag/v4.0). In this guide we will take a look together how to optimize it for improved page speed by picking the fonts we need and reducing the font size.

## Downloading the font

Let's visit the [GitHub Release Page](https://github.com/rsms/inter/releases/tag/v4.0) and download the zip file [Inter-4.0.zip](https://github.com/rsms/inter/releases/download/v4.0/Inter-4.0.zip).

After extracting the zip file, we can find the web fonts int the web folder and the Open Type fonts in the extras/otf folder.

## Downloading the font editing software

For this guide, we are using the open source font editing software Fontforge. You can download it here: [https://fontforge.org/en-US/downloads/](https://fontforge.org/en-US/downloads/)
