# Veryfront MCP

Bring planning, building, and deployment capabilities into your favorite AI coding assistant. Access Veryfront agents directly from Claude, Cursor, or any MCP-compatible environment.

**Good For:** Existing Tool Integration, Power Users

### Installation

1. [Generate an API Key](https://veryfront.com/dashboard/api-keys)
2. Add to your MCP settings:

<CodeBlock showHeader={false}>
  ```json
  {
    "mcpServers": {
      "veryfront": {
        "url": "https://api.veryfront.com/mcp",
        "apiKey": "YOUR_API_KEY"
      }
    }
  }
  ```
</CodeBlock>
