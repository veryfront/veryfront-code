# Veryfront CLI

Command-line tool that integrates agents directly into your terminal and development workflow. Perfect for automation, scripting, and seamless CI/CD pipeline integration.

**Good For**: Professional Coders, Automated Workflows, CI/CD Pipelines

### Installation

1. [Generate an API Key](https://veryfront.com/dashboard/api-keys)
2. Add to `.env` in project root as `VERYFRONT_API_KEY`
3. Run CLI

<CodeBlock showHeader={false}>
  ```bash
  npx @veryfront/cli
  ```
</CodeBlock>
