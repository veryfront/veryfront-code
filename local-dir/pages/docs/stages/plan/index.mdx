# Plan Stage

Transform ideas into plans, broken down as milestones and actionable tasks

| **Type** | **Details**                                                                                                       |
| -------- | ----------------------------------------------------------------------------------------------------------------- |
| Agent    | Planning Agent                                                                                                    |
| Role     | Strategic project planner who transforms high-level concepts into structured milestones and implementation plans. |
| Input    | Ideas, requirements, links, screenshots                                                                           |
| Output   | Milestones and issues documents                                                                                   |

#### Example

**Input**

<CodeBlock showHeader={false}>
  ```prompt
  /plan

  Build a task management app for freelancers, featuring:
  - Project creation and client management  
  - Time tracking with timer functionality
  - Invoice generation based on tracked hours
  - Dashboard showing earnings and project status

  Check @uploads/my-folder for additional context about my idea
  ```
</CodeBlock>

**Output**

```
docs/
├── milestones/
│   └── freelancer-task-management-app.md
└── issues/
    ├── 001-project-client-management-system.md
    ├── 002-time-tracking-timer-functionality.md
    └── 003-invoice-generation-dashboard.md
```

### Tools

**Read**

- `listFiles` - List project files with optional pattern filtering
- `getFiles` - Read content of multiple project files
- `getFile` - Read content of a single project file
- `searchCode` - Search codebase for patterns, symbols, or text
- `listUploads` - List project uploads (images, PDFs, references)
- `fetchUrl` - Fetch content from any remote URL

**Write**

- `createFile` - Create a markdown document
- `updateFile` - Edit an existing document
- `moveFile` - Move or rename a document
- `deleteFile` - Delete a document

### Features

- Convert high-level ideas and requirements into structured implementation plans
- Break down concepts into milestones and actionable tasks
- Persist planning documents for reference and collaboration
- Iterate on plans based on feedback and changing requirements
