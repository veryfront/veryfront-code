# Building Apps with Veryfront

Veryfront accelerates development using a structured process powered by AI agents, to transform ideas into working apps through three stages:

1. **[Plan](#1-plan-stage)**: Turn ideas into structured implementation plans
2. **[Build](#2-build-stage)**: Convert plans to code with previews
3. **[Deploy](#3-deploy-stage)**: Deploy code to production environments

```mermaid
flowchart LR
    A[Idea] --> B[1. Plan] --> C[2. Build] --> D[3. Deploy] --> E[Working App]
```

Start developing in Veryfront using your tool of choice

- [Veryfront Studio](./products/studio/README.md): In-Browser IDE
- [Veryfront CLI](./products/cli/README.md): For Developers and CI/CD
- [Veryfront MCP](./products/mcp/README.md): Integrate with Claude, Cursor etc.
