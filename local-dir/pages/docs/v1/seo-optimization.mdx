### Add SEO title and meta description

The SEO title and meta description are displayed on search engines like Google when users search for your website. To modify the SEO settings for individual pages, you can utilize the [Head component from the core library](https://core.veryfront.com/?path=components\&component=other/Head).

<Alert>
  <info className="h-4 w-4" />
  <AlertTitle>Note</AlertTitle>

  <AlertDescription>
    Once you have made changes to your SEO information, don't forget to publish your site so that the modifications can be implemented.
  </AlertDescription>
</Alert>

```jsx
import { Head } from 'https://core.veryfront.com/@components/index'

<Head>
  <title>Welcome to My Website - Best Deals on Electronics</title>
  <meta name="description" content="Find the best deals on a wide range of electronics, including smartphones, laptops, and smart home devices." />
</Head>
```

### Hide your project from search engines

If you are just starting, you might want to hide your project from search engines until you are ready.

1. To access the project settings, locate and click the gear icon in the sidebar.
2. Disable the `Indexable by search engines` toggle.
3. Click the `Update Project` button.

By performing these steps, you will accomplish the following:

- The robots.txt file will be configured to send a disallow signal to search engines.
- A `<meta name="robots" content="noindex nofollow" />` tag will be added to your page, further indicating to search engines that they should not index or follow the page.
- The sitemap (/sitemap.xml) will be disabled.
- The feed (/feed) will also be disabled.

### Hide one of your pages from search engines

If you want to hide one of your pages from search engines, you can use the [Head component from the core library](https://core.veryfront.com/?path=components\&component=other/Head).

```jsx
import { Head } from 'https://core.veryfront.com/@components/index'

<Head>
  <meta key="robots" name="robots" content="noindex nofollow" />
</Head>
```
