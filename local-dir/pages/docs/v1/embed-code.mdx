<H1>Embed Your App in an Existing Site</H1>

To embed your app in another site, please follow these steps:

1. Go to the embeds tab in the project settings.
2. Enable the option for "Allow your project to be embedded on other sites."
3. Copy the provided embed code. Ensure that your project is published before embedding it.
4. The embed code can be pasted into the HTML of the desired page on the target site.

Additionally, you can customize the embed behavior using the following advanced options:

- URL: Use the URL field to specify the page to load within your app.
- Dimensions: Use the dimensions settings to adjust the height and width of the embedded app.
  - Initial: Set the initial dimensions for the embed, which expands to content dimensions after loading. This setting is useful to set approximate dimensions to avoid flickering on first load.
  - Fixed: Specify fixed dimensions to maintain a consistent size for the embedded app.

After completing these steps and embedding the code, your app will be accessible within the designated page on the target site.
