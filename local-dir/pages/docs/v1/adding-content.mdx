  Adding page content
</H1>

Veryfront's Code Editor allows you to create content using [MDX](https://mdxjs.com) (Markdown with embedded React components). Here's an example of how to add content:

```mdx
import { Button } from "@/shared/ui/Button"
import { Alert, AlertDescription, AlertTitle } from "@/shared/ui/Alert"

# Getting Started

This is an example of using MDX to create rich, interactive content. You can mix Markdown and JSX syntax seamlessly.

## Markdown Features

You can use all the familiar Markdown syntax to format your text:

- **Bold**
- *Italic*
- `Code`
- [Links](https://example.com)
- Lists
  - Item 1
  - Item 2


## JSX Components

MDX allows you to embed JSX components directly in your Markdown. Here's an example:

<Button onClick={() => alert('Hello')}>My Button</Button>
```

In this example, we've added markdown content and embeded a JSX component called "Button".
