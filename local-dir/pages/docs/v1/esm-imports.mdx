<H1>Import ESM modules from npm</H1>

Veryfront allows you to use remote ESM packages in your MDX files. This allows you to use any ESM package from npm directly in your app.

### What is an ESM module?

An ESM (ECMAScript) module is a relatively recent addition to the JavaScript language specification that seeks to unify and standardize how modules are loaded in JavaScript applications. It’s supported by all modern browsers and is the recommended way of writing modular code for the web.

### Supported ESM CDNs

Following ESM CDNs are supported:

- [https://esm.sh](https://esm.sh)
- [https://esm.sh](https://esm.sh)
- [https://cdn.skypack.dev](https://cdn.skypack.dev)
- [https://unpkg.com](https://unpkg.com)
- [https://www.jsdelivr.com](https://www.jsdelivr.com)
- [https://jspm.org](https://jspm.org)

### Example

This example shows how to use the react-confetti package from npm loaded via a remote url:

```mdx
import ReactConfetti from "https://esm.sh/react-confetti"

# Confetti without the cleanup

<ReactConfetti />
```
