<H1>Creating API routes</H1>

1. Open the Sidebar by clicking on the Functions menu icon.
2. To create a new API route, plus icon in the top right corner of the menu.
3. Enter a name and of your API route.
4. If desired, you can customize the slug, where the API route should be accessible.
5. Confirm by clicking "Create Function".
6. The new API route will open in the Code Editor with a default function.
