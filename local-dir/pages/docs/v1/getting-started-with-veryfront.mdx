### Sign-in or sign-up

1. Visit the Veryfront login page at [https://veryfront.com/login](https://veryfront.com/login).
2. Sign in using your Google or GitHub account.
3. If you don't have an account, you can sign up by visiting [https://veryfront.com/sign-up](https://veryfront.com/sign-up). Follow the instructions on the sign-up page to create a new account.
4. Once signed in, you'll be redirected to the Dashboard, where you can manage your projects.

![](https://cdn.veryfront.com/5b74466b-ad05-4594-82f6-8ed08ad1c37c/docs/getting-started-with-veryfront/sign-in.png)

### Create a new project

1. In the Dashboard, click on the "Create Project" button.
2. Provide a name for your project.
3. Click "Create Project" to confirm.
4. You'll then be redirected to the Builder, the main editing view.

![](https://cdn.veryfront.com/5b74466b-ad05-4594-82f6-8ed08ad1c37c/docs/getting-started-with-veryfront/create-project.gif)

### Create your first page

1. In the Builder, you'll find the Code Editor on the left and a Live Preview on the right.
2. To create a new page, click on the Pages menu icon in the Sidebar.
3. Click the plus icon in the top right corner of the Pages menu.
4. Enter a page name and a unique page path.
5. Confirm by clicking "Create Page." The new page will open in the Code Editor.

![](https://cdn.veryfront.com/5b74466b-ad05-4594-82f6-8ed08ad1c37c/docs/getting-started-with-veryfront/create-a-page.gif)

### Add page content

Veryfront's Code Editor allows you to create content using MDX (Markdown with embedded React components). Here's an example of how to add content:

```
import { Button } from "@/shared/ui/Button"

# Getting Started

This is an example of using MDX to create rich, interactive content. You can mix Markdown and JSX syntax seamlessly.


## Markdown Features

You can use all the familiar Markdown syntax to format your text:

- **Bold**
- *Italic*
- `Code`
- [Links](https://example.com)
- Lists
  - Item 1
  - Item 2


## JSX Components

MDX allows you to embed JSX components directly in your Markdown. Here's an example:

<Button onClick={() => alert('Hello')}>My Button</Button>
```

In this example, we've added markdown content and embeded a JSX component called "Button".

![](https://cdn.veryfront.com/5b74466b-ad05-4594-82f6-8ed08ad1c37c/docs/getting-started-with-veryfront/add-page-content.gif)

### Save the page

Once you're satisfied with your page, save it by using the <kbd className="bg-secondary-foreground text-white dark:text-black py-1 px-2 rounded">CMD</kbd> + <kbd className="bg-secondary-foreground text-white dark:text-black py-1 px-2 rounded">S</kbd> / <kbd className="bg-secondary-foreground text-white dark:text-black py-1 px-2 rounded">CTRL</kbd> + <kbd className="bg-secondary-foreground text-white dark:text-black py-1 px-2 rounded">S</kbd> shortcut.

Saving creates a snapshot of your page with a version number, allowing for easy rollback if needed.

### Publish your app

1. To publish your app, locate the "Publish" button in the upper right corner of the Top Bar in the Builder.
2. Click on the button, and after publishing, an overlay will display the link to your app in production.
3. Open the provided link in a new browser tab to view your published app.
4. Update the URL with the page's path to see your newly created page.

Congratulations! You have successfully published your first app with Veryfront.

![](https://cdn.veryfront.com/5b74466b-ad05-4594-82f6-8ed08ad1c37c/docs/getting-started-with-veryfront/save-publish.gif)
