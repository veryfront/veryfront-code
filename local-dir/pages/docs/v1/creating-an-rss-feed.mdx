<H1>Creating a RSS Feed using API Routes</H1>

This guide will walk you through the process of creating a RSS Feed for your web app built on Veryfront. We'll use Veryfront's API routes and Node.js to accomplish this task.

### Create an API route

Create an API route for serving your RSS Feed. To learn how to create an API route checkout the [Creating API Routes](/docs/api-routes) section.

### Add the Code for the RSS Feed API Route

1. Inside the API route you created, use the Veryfront API to query all the pages of your web app. This can be done by making a `pages` GraphQL API call to retrieve the necessary data.
2. Once you have the list of pages, you'll need to transform it into XML format, which is the standard format for RSS Feeds. This can be accomplished by iterating over the pages and generate the appropriate XML elements.
3. Once the XML string is populated, you need to serve it to search engines and other interested parties.
4. Make sure the API route returns the generated XML as the response body and that the appropriate HTTP headers are set.

You can find the full code including comments in the next section.

import got from "https://esm.sh/got@12.6.1?target=node";

export default async function ({ req }) {

  // GraphQL query to fetch pages data
  let json = {
    query: `query PagesListQuery($first: Int, $cacheKey: String!, $filter: FilterParams) {
    pages(first: $first, cacheKey: $cacheKey, filter: $filter) {
      edges {
        cursor
        node {
          slug
          isPublished
          frontmatter
        }
      }
    }
  }`,
    variables: {
      first: 1000,
      filter: {
        sort: "name:ASC"
      },
      "cacheKey": process.env.VERYFRONT_PROJECT_ID
    }
  }

  // Send a POST request to the GraphQL API
  const { data } = await got
    .post("https://api.veryfront.com/graphql", {
      headers: {
        'x-project': process.env.VERYFRONT_PROJECT_ID,
        'Content-Type': 'application/json',
      },
      json,
    })
    .json();

  const baseUrl = 'https://veryfront.com/';

  // Channel metadata for the RSS feed
  const channel = {
    title: "Veryfront",
    feedUrl: "https://veryfront.com/api/rss",
    language: "en",
    image: "https://cdn.codersociety.com/images/veryfront-og-image-logomark.png",
    description: "On our blog we share our take on the latest technology trends, and practical guides on new technologies and methods.",
  }

  // Generate the channel image tag if available
  const channelImage = channel.image
    ? `<image>
        <title>${channel.title}</title>
        <url>${channel.image}</url>
        <link>${baseUrl}</link>
      </image>`
    : ''

  // Filter pages that have RSS enabled and map them to feed items
  const feedItems = data?.pages?.edges?.filter(({ node }) => node?.frontmatter?.rss)
  
  // Generate the channel feed items based on the filtered pages
  const channelFeed = feedItems?.map(({ node }) => {
    const link = baseUrl + (node?.slug === '/' ? '' : node?.slug)
    const meta = node?.frontmatter?.context?.meta || {}
    const title = meta.title
    const description = meta.description
    const publicationDate = meta.publicationDate
    const image = meta.image
    const creator = meta.author || channel.title
    const imageTag = image ? `<img src="${image}" alt="${title || ''}" />` : ''

    return `
      <item>
        <title><![CDATA[${title}]]></title>
        <link>${link}</link>
        <guid isPermaLink="true">${link}</guid>
        ${publicationDate ? `<pubDate>${new Date(publicationDate).toUTCString()}</pubDate>` : ''}
        <dc:creator><![CDATA[${creator}]]></dc:creator>
        ${description ? `<description><![CDATA[${description}]]></description>` : ''}
        ${ imageTag ? `<content:encoded><![CDATA[${imageTag}]]></content:encoded>` : ''}
      </item>
    `
  }).join('')

  // Generate the RSS feed XML
  const rssFeed = `<?xml version="1.0" encoding="UTF-8"?>
  <rss version="2.0" xmlns:content="http://purl.org/rss/1.0/modules/content/" xmlns:dc="http://purl.org/dc/elements/1.1/" xmlns:atom="http://www.w3.org/2005/Atom">
    <channel>
      <title>${channel.title}</title>
      <link>${baseUrl}</link>
      <atom:link href="${channel.feedUrl}" rel="self" type="application/rss+xml" />
      ${channel.description ? `<description>${channel.description}</description>` : ''}
      ${channel.updatedAt ? `<lastBuildDate>${new Date(channel.updatedAt).toUTCString()}</lastBuildDate>`: ''}
      <language>${channel.language}</language>
      ${channelImage}
      ${channelFeed}
    </channel>
  </rss>
  `

  // Return the RSS feed XML as the response
  return {
    headers: {
      'Content-Type': 'application/xml',
      'Content-Length': Buffer.byteLength(rssFeed),
    },
    body: rssFeed,
  }
}
