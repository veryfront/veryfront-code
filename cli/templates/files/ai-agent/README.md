# AI Agent

A simple conversational AI with tool support.

## What's included

- Single assistant agent with streaming chat UI
- Example calculator tool
- `useChat` hook for real-time responses

## Structure

```
agents/assistant.ts    Agent definition
tools/calculator.ts    Example tool
app/
  api/ag-ui/route.ts    AG-UI endpoint
  page.tsx             Chat interface
```

This starter is not production-ready.
