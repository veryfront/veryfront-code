# Getting Started

This guide will help you get up and running with your new project.

## Prerequisites

- Node.js 18+
- npm or yarn

## Installation

1. Create a new project:

```bash
veryfront init my-app
cd my-app
```

2. Install dependencies:

```bash
npm install
```

3. Start the development server:

```bash
veryfront dev
```

## Project Structure

```
my-app/
  app/
    page.tsx         # Home page
    layout.tsx       # Root layout
    docs/
      page.mdx       # Documentation page
  public/
    robots.txt       # SEO configuration
  veryfront.config.js # Configuration
```
